Слайдер с двумя тогглерами.

Структура:
- /source : исходные файлы проекта.
- /build : готовая сборка.

Слайдер реализован отдельным модулем. Для подключения к проекту необходимо импортировать функцию initRange().

import { initRange } from './range.js';

Функция initRange принимает следующие параметры:
- selector - селектор элемента. По умолчанию "#range-wrapper".
- minValue - минимальное значение, которое сможет принимать слайдер.
- maxValue - максимальное значение, которое сможет принимать слайдер.
- minDefault - значение левого тогглера по умолчанию (при загрузке страницы).
- maxDefault - значение правого тогглера по умолчанию (при загрузке страницы).

При необходимости изменить параметры достаточно повторно вызвать initRange с новыми параметрами.
