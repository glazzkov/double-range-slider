Спаренный range с двумя тогглерами.

Структура:
- /source : исходные файлы проекта.
- /build : готовая сборка.

1. Слайдер реализован отдельным модулем. Для подключения к проекту необходимо импортировать функцию range().

  import { range } from './range.js';

2. Далее необходимо создать переменную и присвоить ей range c начальными параметрами.

  Функция range принимает следующие параметры (имеются значения по умолчанию):
    - selector - селектор элемента. По умолчанию "#range-wrapper".
    - minValue - минимальное значение, которое сможет принимать слайдер.
    - maxValue - максимальное значение, которое сможет принимать слайдер.
    - minDefault - значение левого тогглера по умолчанию (при загрузке страницы).
    - maxDefault - значение правого тогглера по умолчанию (при загрузке страницы).

  Пример вызова функции
    const range1 = range('#range-wrapper', 80, 500, 200, 450);

3. После вызова range необходимо его инициализировать:

  range1.init();

4. range подключен к странице.

5. На странице index.html в качестве примера подкючено сразу 2 range с разными параметрами.

6. При необходимости изменить параметры достаточно повторно вызвать функцию init с новыми параметрами.
